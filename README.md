# DOOM-RL Projekt

## Umgebung
### 1. Virtuelle Umgebung erstellen
python -m venv venv

### 2. Aktivieren
Windows: 
```bash
venv\Scripts\activate
```
Mac: 
```bash
source venv/bin/activate
```

### 3. Pakete installiere
```bash
pip install -r requirements.txt
```

### 4. Requirements speichern
Wenn du neue Pakete installiert hast:
```bash
pip freeze > requirements.txt
```
